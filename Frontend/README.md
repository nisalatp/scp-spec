# Frontend Guide (React)

This folder contains a comprehensive guide for React frontend developers working on SCP.
It is written for new team members with limited experience.

## How to use this guide

1. Start with `01-overview.md`.
2. Follow the files in order.
3. Use the checklists at the end of each file.
