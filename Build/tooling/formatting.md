# Formatting Standards

SCP uses automated formatting to ensure code readability and reduce diff noise.

## Generic
- **Indentation**: 2 spaces.
- **Line Length**: Max 120 characters.
- **End of Line**: LF (not CRLF).

## Frontend
- **Prettier**: Configured in `.prettierrc`.
- **Imports**: Alphabetical sorting, grouped by (External, Internal, Styles).

## Backend
- **IDE Formatter**: Use standard C#/Java/JS formatting templates.
- **Naming**: PascalCase for classes/methods, camelCase for variables/fields.
