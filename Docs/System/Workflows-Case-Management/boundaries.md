# Boundaries: Workflows & Case Management

## Ownership
- **Owns**: The "State" of the beneficial relationship.
- **Owns**: The history of approvals.

## Never Owns
- **The Policy**: The *Eligibility Rules* determine if someone *should* pass. The Workflow Engine *executes* that check.

## Integration Boundaries
- **Inbound**:
    - **Form Builder**: Forms initiate Cases.
- **Outbound**:
    - **Finance**: Active Cases trigger payments.
