# Programs

## Purpose
The Programs component defines the structured initiatives and social protection interventions that the platform delivers. It acts as the high-level policy container that link vertical governance, eligibility rules, and case management.

## Responsibilities
- **Program Definition**: Managing the lifecycle of initiatives.
- **Scoping**: Defining geographical and chronological boundaries.
- **Vertical Alignment**: Managing primary and participant vertical relationships.
