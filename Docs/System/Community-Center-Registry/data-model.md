# Data Model: Community Center Registry

## Entities

### CommunityCenter
- `center_id`
- `name`
- `type` (Hall, School, Temple, Church, Mosque)
- `geo_node_ids` (List of Villages served)
- `owner_id` (Person ID of custodian)
- `verification_status` (Pending, Verified)
