# Workflows: Community Center Registry

## Workflow: Registering a Village Hall
1.  **Community Leader**: Submits "Village Hall" via Public App.
2.  **Status**: `Draft`.
3.  **Field Agent**: Receives Task "Verify Hall".
4.  **Field Agent**: Visits, takes photo, checks GPS. Confirms.
5.  **System**: Updates to `Verified`.
6.  **Usage**: Now selectable for "Vaccination Drive Location".
