# Identity Features

This directory documents the user-visible and operator-visible features of the Identity component.

## Features
- **[Register Person](register-person.md)**: Enrolling new beneficiaries.
- **[Verify Identity](verify-identity.md)**: Confirming provisional records.
- **[Update Person Profile](update-person-profile.md)**: Corrections and Life Events.
- **[Search Person](search-person.md)**: Finding records.
- **[Archive / Restore Person](archive-restore-person.md)**: Soft deletion and recovery.
