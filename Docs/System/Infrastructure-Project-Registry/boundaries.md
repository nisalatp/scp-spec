# Boundaries: Infrastructure Project Registry

## Ownership
- **Owns**: The *Metadata* of the project (Definition, Phase).
- **Owns**: The *Map* of where the project is happening.

## Never Owns
- **The Asset**: Once built, a "School" becomes a `Facility`. The *Project* to build it closes, and the *Facility* opens.

## Integration Boundaries
- **Inbound**:
    - **Planning Ministry**: Uploads 5-Year Plan projects.
- **Outbound**:
    - **Public Dashboard**: Shows map of "Where is my tax money going?".
