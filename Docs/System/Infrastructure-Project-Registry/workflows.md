# Workflows: Infrastructure Project Registry

## Workflow: Planning a New Road
1.  **Planner**: Defines "Highway Extension A1".
2.  **Scope**: Selects 10 GN Divisions the road passes through.
3.  **Approval**: Ministry signs off. Status -> `Active`.
4.  **Visibility**: Residents in those 10 Divisions see "Active Roadwork" in their community app.

## Workflow: Completion
1.  **Manager**: Marks Status -> `Completed`.
2.  **System**: Archives the active project record (moves to History).
