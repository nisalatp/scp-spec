# Workflows: Case Worker

## Use Cases
*   **Application Review**: Processing a pending application.
*   **Case Closure**: Closing a case after support ends.

## Scenario: Verification

1.  **Queue**: Worker sees list of "Submitted" visits in their dashboard.
2.  **Review**: Opens Visit #123. Checks ID photos.
3.  **Action**: Clicks "Verify".
4.  **Result**: Visit status -> Verified. Data committed to Person Profile.
