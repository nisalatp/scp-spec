# Workflows: Field Agent

## Use Cases
*   **New Registration**: Adding a household that moved in.
*   **Update**: Correcting a phone number.

## Scenario: Delivery

1.  **Arrive**: Agent arrives at House 4.
2.  **Identify**: Asks for ID card. Scans/Types ID.
3.  **Result**: App shows "Eligible for Food Pack".
4.  **Action**: Hand over pack. Record "Delivered".
5.  **Evidence**: Take photo of recipient.
