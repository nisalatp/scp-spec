# Role: Geo Admin

## Overview
The **Geo Admin** (often the Surveyor General's office) is responsible for maintaining the Master Location Data of the platform. They define the administrative hierarchy (Provinces, Districts) and ensure that all System Entities are correctly mapped to a location.

## Key Responsibilities
- Defining the Geo Hierarchy structure.
- Creating and updating Geo Nodes (Villages, Divisions).
- Managing Geo Splits and Merges.
- Defining Geo Metrics (e.g., Risk Indices).

## Critical Permissions
- `geo.hierarchy.manage`
- `geo.node.create`
- `geo.node.update`
- `geo.metric.manage`
- `geo.reorg` (High Privilege)

## Known Users
- Ministry of Home Affairs (Survey Dept).
- National Statistics Office.
