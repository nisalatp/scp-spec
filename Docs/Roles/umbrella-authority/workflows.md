# Workflows: Umbrella Authority

## Use Cases
*   **Program Approval**: Reviewing a new "Maternal Health" program proposed by the Health Vertical.
*   **Policy Update**: Approving a change to data retention policies.
*   **Conflict Resolution**: Deciding which household record is authoritative in a duplicate dispute.

## Scenario: Program Approval

1.  **Notification**: Umbrella receives request for "School Lunch Program" (Education Vertical).
2.  **Review**: Checks budget, KPIs, and eligibility rules.
3.  **Action**: Clicks "Approve".
4.  **Result**: Program status changes to "Approved". It can now be activated by the Vertical Owner.
